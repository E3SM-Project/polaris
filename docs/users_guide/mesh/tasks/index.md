# Tasks

## Spherical Mesh Tasks

These tasks operate on spherical MPAS meshes without being specific to the
unified-mesh workflow.

```{toctree}
:titlesonly: true

base_mesh
feature_masks
```

## Unified Mesh Tasks

These tasks cover the full unified-mesh workflow for named global meshes:
coastline and river preprocessing, sizing-field construction, and final
MPAS base-mesh generation.

```{toctree}
:titlesonly: true

unified/coastline
unified/river
unified/sizing_field
unified/base_mesh
```
