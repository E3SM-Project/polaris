timestep and run duration

The test runs with a one hour timestep using 2000-01-01 forcing. A ‘full_run’ is run for one day with restarts written after 12 hours and 24 hours. A second ‘restart_run’ is run for 12 hours, starting from the 12 hour restart from the ‘full_run’. The contents of the two restart files written at the end of the pair of runs are then compared.
