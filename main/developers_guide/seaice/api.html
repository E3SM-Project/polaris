seaice

SeaIce()

The collection of all test case for the MPAS-Seaice component

Test Groups

single_column

SingleColumn(component)

A test group for "single column" test cases

forward.Forward(test_case[, name])

A step for staging a mesh for “single column” test cases

exact_restart.ExactRestart(test_group)

A restart test case for the single column test group, which makes sure the model produces identical results with one longer run and two shorter runs with a restart in between.

exact_restart.ExactRestart.validate()

Compare variables in the restart files at the end of each run.

standard_physics.StandardPhysics(test_group)

The standard physics test case for the "single column" test group creates the mesh and initial condition, then performs a short forward run.

standard_physics.StandardPhysics.validate()

Compare six output variables in the forward step with a baseline if one was provided.

standard_physics.viz.Viz(test_case)

A step for plotting the results of a single column test

standard_physics.viz.Viz.run()

Run this step of the test case
